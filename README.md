# 🖨️ TIS Service v2.0

HP Printer PDF to Excel with Paddle OCR (96-98% Accuracy)

## Features

- ✅ Paddle OCR - 96-98% accuracy
- ✅ 6 HP Models - M404, M406, M428, M4003, M4103, M430
- ✅ Fast - 8-10 seconds
- ✅ Free - Local OCR
- ✅ Thai Support

## Quick Start

```bash
npm install
npm run dev
```
