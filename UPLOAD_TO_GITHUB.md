# 📤 Upload to GitHub

## วิธีอัปโหลดแบบง่ายที่สุด

### Step 1: Create GitHub Repo
1. ไป https://github.com/new
2. Repository name: `tis-service`
3. Public ✓
4. Create

### Step 2: Upload Files
1. ใน GitHub repo คลิก **"Add file"** → **"Upload files"**
2. Drag & drop หรือเลือก ไฟล์ทั้งหมดจากโฟลเดอร์นี้
3. สามารถอัปโหลด folder และ files พร้อมกัน
4. Commit

### Step 3: Done! ✅
Repository live at: `github.com/YOUR_USERNAME/tis-service`
